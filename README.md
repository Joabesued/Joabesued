## 🚀 Minha Jornada / *My Journey*

### 🥿 Chico Sapateiro — Sistema de Gestão
**Sistema web completo para modernizar a sapataria da família**

Do caderno à nuvem. Construí do zero um sistema de gestão completo para substituir o controle manual em caderno que meu pai usava há décadas.

*From notebook to cloud. I built from scratch a complete management system to replace the manual control my father had been using for decades.*

- ⚙️ **Stack:** React + Tailwind CSS · Python + FastAPI · PostgreSQL (Supabase)
- ☁️ **Deploy:** Vercel (frontend) + Railway (backend)
- 📱 **Features:** Ordens de serviço · Checklist · WhatsApp integrado · Relatório financeiro · Área Admin · Ditado por voz · Foto por item
